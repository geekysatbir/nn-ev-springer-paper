# Neural Networks and Electric Vehicles - Springer Publication

## Publication Details
This repository contains the research paper on Neural Networks and Electric Vehicles published by Springer.

## Abstract
The development of many control strategies to enhance the performance of EVs, mainly with regard to energy control, power efficiency and overall control systems, has emerged as a result of the rising demand for EVs. This review paper focuses on the possible application of the hybrid neural network and fuzzy logic control for enhancing some performance characteristics of EVs.

## Code Implementation

This repository includes a Python implementation of the hybrid neural network and fuzzy logic control system described in the paper.

### Features
- **Fuzzy Logic Controller**: Rule-based decision making for real-time motor control
- **Neural Network Predictor**: Deep learning model for energy consumption prediction
- **Hybrid EV Controller**: Integration of both approaches for optimal performance

### Installation

```bash
pip install -r requirements.txt
```

### Usage


### Key Components

1. **FuzzyLogicController**: Implements fuzzy membership functions and inference engine
2. **NeuralNetworkPredictor**: Deep neural network for energy prediction
3. **HybridEVController**: Combines both approaches for optimal energy management
